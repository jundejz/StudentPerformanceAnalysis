Analysing Performance by Gender

We start off our analysis by creating 2 new tables, using the code below, from our original table, so that we can seperate the scores of men and women.

create table men_all_scores select * from all_scores where gender = 1; create table women_all_scores select * from all_scores where gender = 0;
Next, using basic aggregate functions we can obtain some metrics.
